About Chitraranga

Chitraranga is an intelligent image colorization platform built using Deep Learning and a powerful U-Net architecture. It takes grayscale images and brings them to life with realistic colors.

The model is trained using TensorFlow and deployed through a Flask web application that allows you to upload, process, and download colorized images instantly.
